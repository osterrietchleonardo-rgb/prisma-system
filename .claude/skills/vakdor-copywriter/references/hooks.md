# Librería de Hooks Parametrizados — Vakdor / PRISMA

Banco de 80 ganchos calificados para el ICP de PRISMA. El hook es la dimensión de mayor apalancamiento: si no para el scroll del ICP correcto, no existe el resto de la pieza.

## ICP al que le hablan estos hooks
Director / presidente / gerente / dueño de inmobiliaria con:
- **+30 asesores** y **+300 propiedades** en cartera.
- Capacidad de inversión (busca un partner de largo plazo, no un servicio barato).
- Dolor global: **dependencia operativa**. De ahí se desprenden los 5 ejes: **control, trazabilidad, sistematización (procesos + conocimiento), sistemas integrales y toma de decisiones informadas**.

## Cómo usar esta librería
1. En el PASO 0 (brief), elegí el hook por **eje** (el sub-dolor que ataca la pieza) y por **tipo**.
2. Reemplazá las **variables** `[...]` con datos reales o del cliente. Nunca dejes el corchete.
3. Estos hooks están diseñados para **LinkedIn (modo ultracualificación)**: llevan calificador de escala y posición fuerte para repeler al no-ICP. Para **Instagram (modo alcance/RUM)**, quitá el calificador de escala y suavizá la polarización.
4. Regla de oro: específico > genérico, número exacto > redondo, concreto > abstracto. El ICP tiene que sentir "esto es para mí" en la primera línea.
5. **Segunda persona SIEMPRE:** hablale directo a él/ella ("vos", "tu agencia", "tenés"), nunca del rubro en general ("el director", "las inmobiliarias"). Único recurso permitido: el contraste "la mayoría… vs vos", donde el golpe igual aterriza en "vos".

## Variables
- `[N]` = porcentaje o número exacto (nunca redondo)
- `[N_asesores]` = cantidad de asesores (umbral real, ej. 30, 50)
- `[N_props]` = propiedades en cartera (ej. 300)
- `[zona]` = mercado/zona (ej. AMBA, Córdoba)
- `[periodo]` = mes / trimestre
- `[monto]` = cifra de dinero

## Campos de cada ficha
`tipo` (clase de gancho) · `fractura` (Hemorragia / Anarquía / Ceguera) · `capa` (1 dolor · 2 miedo · 3 deseo · 4 sueño) · `disparador` (psicológico) · `objetivo` (O1–O10) · `calificador` (explícito / implícito)
