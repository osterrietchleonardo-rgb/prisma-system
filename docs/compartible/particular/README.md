# Documentación particular por cliente — INTERNO (no se sube al repo)

Acá vive la copia personalizada de las guías funcionales para cada agencia que tenga
ajustes propios (módulos habilitados/deshabilitados, condiciones particulares, etc.).

## Convención

- **Una subcarpeta por cliente:** `nombre-cliente[-agencia]/` (ej: `kevin-arlandi/`).
- Adentro, las dos guías:
  - `FUNCIONAL-ASESOR-PRISMA.md`
  - `FUNCIONAL-DIRECTOR-PRISMA.md`
- Se **parten de la versión estandarizada** (`../estandarizada/`) y se ajusta únicamente lo
  específico de esa agencia (típicamente la sección "Personalizaciones de tu agencia").

## Importante

- Todo lo que esté dentro de `particular/` está **gitignoreado** (salvo este README).
  Contiene información de clientes y **NO debe subirse al repositorio**.
- Mantené las versiones de cliente sincronizadas con la estandarizada cuando esta cambie.
